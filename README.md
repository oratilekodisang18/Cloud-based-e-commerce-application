# Cloud-Based E-Commerce Application

This project is a cloud-hosted web application developed using Microsoft Azure, designed around an online retail scenario.

## Overview
The application demonstrates the use of cloud-native services to store customer and product data, process orders, and support scalability and reliability.

## Key Features
- Cloud-based data storage for products and customers
- Asynchronous processing using queue-based messaging
- Serverless functionality for background tasks
- Deployed web application using Azure App Service

## Technologies Used
- Microsoft Azure
- Azure App Service
- Azure Functions
- Azure Blob Storage
- Azure Table Storage
- Azure Queue Storage
- Azure SQL Database
- GitHub

This project showcases practical experience with cloud architecture, scalable systems, and Azure services.
